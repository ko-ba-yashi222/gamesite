SYSTEM

Basic Class

本作ではクラスシステムを採用しており、プレイヤーがキャラクターの役割や戦闘スタイルを選ぶことができます。
各クラスは異なる能力、スキル、戦術を持ち、プレイスタイルに合わせて成長します。
戦士として戦場で仲間を支える盾となるか、魔法使いとして強力な呪文で敵を一掃するか、 暗殺者となり影から戦況を操るもよし、僧侶となり神聖な力で味方を支援するか選択はプレイヤー次第。
ゲーム開始時には複数の基本クラスから選び、進行に伴い上級クラスへと進化することができます。 また、クラスにはスキルツリーやカスタマイズ要素があり、自分のスタイルに合わせて成長を楽しむことができます。

Warrior

ウォリアーは近距離戦闘に優れたクラスで、堅固な防御力と高い攻撃力を誇ります。鋭い剣や盾を駆使し、敵の前線で仲間を守りながら戦うことができます。特に耐久力が高く、ダメージを受けても立ち続けることができるため、パーティの盾となる役割を担います。

Sorcerer

ソーサラーは遠距離から強力な魔法で敵を攻撃するクラスです。
火、氷、雷などの元素魔法を駆使し、戦場に大きな影響を与えることができます。高い攻撃力を誇る反面、耐久力は低いため、安全な位置を確保することが重要となります。

Assassin

アサシンは中距離・近距離に対応できるクラスです。
影のように忍び寄り、瞬時にターゲットを仕留めます。高い敏捷性と隠密行動を活かし、敵に気づかれずに致命的な一撃を加えることができます。

Priest

プリーストは神聖な力を使って仲間を癒し、サポートするクラスです。
癒しの魔法や祝福、守護の呪文を使い、戦闘中の仲間を支援します。攻撃力は低いものの、その癒しの力は絶大で、パーティの生命線ともいえる存在です。

Skill Tree

多様なビルドの可能性
スキルツリーを活用することで、攻撃的なビルド、支援型のビルド、あるいは防御に特化したビルドなど、プレイヤーごとのプレイスタイルに合わせたキャラクターを作成可能です。複数のスキルパスが存在するため、他のプレイヤーと差別化された独自のキャラクターを作り上げることができます。

スキルツリーの進化
さらに、スキルツリーは単なる成長の道筋にとどまらず、進化の選択肢も用意されています。一定の条件を満たすことで、上級スキルを習得できる「進化ルート」に進むことができ、プレイヤーの選択がキャラクターの成長に大きな影響を与えます。

プレイスタイルに合わせた最適化
プレイヤーはスキルツリーを使って、PvE（プレイヤー対環境）やPvP（プレイヤー対プレイヤー）において、自分の目的に最適なビルドを作り上げることができます。スキルを組み合わせることで、ダンジョン攻略や対人戦において強力なパフォーマンスを発揮することが可能です。

スキルツリーを制覇し、自分だけの戦術で世界を制覇しましょう！

必要・推奨動作環境

必要動作環境

  • OS（オペレーティングシステム:
    Windows 7 以上 または macOS 10.13（High Sierra）以上
  • プロセッサ(CPU):
    Core i3-6100 または AMD FX-4100 以上
  • グラフィックカード(GPU):
    NVIDIA GeForce GTX 750 Ti 以上 または AMD Radeon R7 360 以上
  • メモリ(RAM)
    8GB 以上
  • ストレージ
    HDD 512gb 以上
  • ネットワーク接続
    標準的なインターネット接続（ダウンロード 5Mbps 以上、アップロード 1Mbps 以上）
  • DirectX / Vulkan
    DirectX 11 以上
  • その他の要件
    サウンドカード、解像度 1280x720 以上、マウス・キーボード

推奨動作環境

  • OS（オペレーティングシステム:
    Windows 10 / 11（64ビット）または macOS 10.15（Catalina）以上
  • プロセッサ(CPU):
    Intel Core i5-9400F 以上 または AMD Ryzen 5 2600 以上
  • グラフィックカード(GPU):
    NVIDIA GeForce GTX 1660 以上 または AMD Radeon RX 580 以上
  • メモリ(RAM)
    16GB 以上
  • ストレージ
    SSD 512GB 以上（ゲームのロード時間を短縮するため、SSDが推奨されます）
  • ネットワーク接続
    高速インターネット接続（ダウンロード 10Mbps 以上、アップロード 3Mbps 以上）
  • DirectX / Vulkan
    DirectX 11 以上
  • その他の要件
    サウンドカード、解像度 1920x1080（フルHD）以上、マウス・キーボード
